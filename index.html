Publications

Here are some of my notable publications:

  1. Divya Thekke Kanapram, Lucio Marcenaro, David Martin Gomez, Carlo Regazzoni
    Graph-Powered Interpretable Machine Learning Models for Abnormality Detection in Ego-Things Network
    [PDF] Sensors, 2022

    This paper introduces a graph-powered interpretable ML model for abnormality detection, integrating self-awareness (SA) and collective awareness (CA) functionalities in agent networks. Using generative Dynamic Bayesian Networks (DBNs) and a Markov Jump Particle Filter (MJPF), interpretability is achieved via graph matching of semantic vocabulary. This approach enables the model to self-explain its decisions and incrementally update learned representations based on anomalies.

  2. Divya Thekke Kanapram, Fabio Patrone, Pablo Marin-Plaza, Mario Marchese, Eliane L. Bodanese, Lucio Marcenaro, David Martín Gómez, and Carlo Regazzoni
    Collective Awareness for Abnormality Detection in Connected Autonomous Vehicles
    [PDF] IEEE Internet of Things Journal, VOL. 7, NO. 5, MAY 2020

    This article develops collective awareness (CA) models for networked autonomous vehicles, enabling synchronous abnormality detection across the system. It employs data-driven DBNs inferred by a Markov Jump Particle Filter (MJPF), utilizing Hellinger distance metrics to quantify anomalies. The significance lies in assessing the model's robustness against realistic communication channel effects, including packet loss and delay via the IEEE 802.11p protocol.

  3. Divya Thekke Kanapram, Pablo Marin-Plaza, Lucio Marcenaro, David Martin, Arturo de la Escalera, and Carlo Regazzoni
    Self-awareness in Intelligent Vehicles: Feature-Based Dynamic Bayesian Models for Abnormality Detection
    [PDF] Robotics and Autonomous Systems Journal, Volume 134, 2020

    This work introduces self-awareness (SA) models for intelligent vehicles, focusing on abnormality detection via feature-based switching DBNs. The method trains DBNs on pairs of low-dimensional sensor data, using an MJPF to identify optimal feature combinations for reliable prediction. This approach achieved high accuracy (AUC 78.20%) in automatically detecting unseen maneuvers using the Steering Angle-Power feature pair.

  4. Divya Thekke Kanapram, Damian Campo, Mohamad Baydoun, Lucio Marcenaro, Eliane L. Bodanese, Carlo Regazzoni, and Mario Marchese
    Dynamic Bayesian Approach for Decision-Making in Ego-Things
    [PDF] 2019 IEEE 5th, World Forum on Internet of Things (WF-IoT)

    This paper presents an approach for abnormality detection in ego-things by generating multiple DBN inference models from diverse sensor data features. Utilizing Growing Neural Gas (GNG) and a Markov Jump Particle Filter (MJPF), the system employs Hellinger distance to assess model precision. The method successfully selects the optimal feature set (Steering Angle-Power) for maximizing anomaly recognition and enhancing prediction accuracy in networking operations.

  5. PhD thesis - Title: Ego things: Networks Of Self-Aware Intelligent Objects. Multi-modal dynamic Bayesian models for self-awareness and collective awareness in agents’ network : The research aimed to develop a hierarchical dynamic Bayesian Network using data-driven methods, incorporating self-awareness and collective awareness in intelligent agent networks for anomaly detection, decision-making, and incremental learning of new models from new experiences. Machine learning, signal processing, and IoT were utilized to implement these functionalities while analyzing the effects of networking protocols and communications. The model was further enhanced with interpretability by applying graph-matching techniques.

  6. A framework to correlate power consumption and resource usage in cloud infrastructures

    This paper presents the Green Cloud Abstraction Agent (GCAA) framework designed to correlate resource usage (e.g., CPU) and energy consumption in cloud infrastructures, supporting energy-efficiency strategies like workload consolidation. The GCAA implementation integrates power measurements gathered via Kwapi and IPMI into OpenStack Ceilometer to provide dynamic, precise power-to-usage curves. This capability is essential for orchestrators optimizing VM placement in both single and federated clouds.

  7. Exploring the trade-off between performance and energy consumption in cloud infrastructures

    This work outlines a framework for exploring the performance-energy consumption trade-off, vital for optimizing workload consolidation in distributed (fog/edge) cloud environments. The proposed Green Cloud Abstraction Agent (GCAA) architecture uses the Kwapi framework to integrate power measurements (via SNMP/IPMI) into OpenStack Ceilometer. This infrastructure provides the necessary energy abstraction for green orchestration, enabling algorithms to make efficient VM placement decisions.
